# 🩺 Medical Case Simulator

An interactive web-based simulator that allows users to practice clinical decision-making by analyzing patient symptoms, selecting appropriate tests, and determining diagnoses.

## 🚀 Features
- Interactive case-based learning
- Test selection validation
- Diagnosis evaluation
- Score tracking system
- Progress tracking

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 🎯 How It Works
1. Analyze patient symptoms
2. Choose the correct diagnostic test
3. Determine the correct diagnosis
4. Receive feedback and score


## 🚀 Future Improvements

- Add a larger and more diverse set of medical cases  
- Introduce difficulty levels (easy, medium, hard)  
- Improve UI/UX with smoother transitions  
- Add a timer-based quiz mode  
- Provide performance analytics (accuracy and progress tracking)  
- Deploy the application for wider accessibility and user feedback


## 📊 Data Source

The medical case scenarios used in this project are manually curated for educational purposes. 
The symptoms, diagnostic tests, and explanations are based on general medical knowledge 
and publicly available information from reliable health resources.

This project does not use real patient data and is intended only for learning and demonstration purposes.

## 📌 Note
This project is built for educational purposes and uses simplified medical cases.
